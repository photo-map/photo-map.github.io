## Develop

Generate `.env` with content:

```
REACT_APP_GOOGLE_MAPS_API_KEY=AIz***kC8
REACT_APP_AMAP_API_KEY=ef0***63e
```

Run dev server.

```
$ yarn && yarn start
```
